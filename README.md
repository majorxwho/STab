## 🛠️ 核心功能与特色

### 1. ⚡ 网页快捷收藏 (Alt + S)
浏览网页时按下 `Alt + S` 即可快速收藏。支持在弹窗内直接新建文件夹并分类，比传统收藏夹更高效、更清爽。

### 2. ☁️ 三通道安全云同步
内置 **Supabase 云数据库**、**NAS 私有云**、**标准 WebDAV**（支持坚果云、Koofr、Yandex Disk 等）三大通道，收藏数据自动双向同步。

### 3. 🌐 多浏览器无缝互通
支持 Chrome、Edge、Firefox 等不同浏览器之间数据实时对齐。一次设置，全端通用。

### 4. 🔒 字符与手势双重防护
文件夹支持“手势轨迹”与“字符密码”双重加密。手势解锁快捷方便，字符密码提供极高安全性。

### 5. 🛡️ 隐私强化机制
加密文件夹可开启：
- **隐藏缩略图**：防止视觉泄露。
- **禁用全局检索**：防止从搜索框意外搜出私密网址。
- **无痕浏览模式**：点击网址自动开启隐私窗口，不留历史痕迹。

### 6. 🧠 智能管理系统
- **自动排重**：添加网址、导入备份时智能识别重复项。
- **无损合并**：同名文件夹自动安全合并，数据不丢失。
- **后台静默多开**：按住 `Ctrl` 键点击卡片即可在后台打开新标签页。

### 7. 🎨 极简视效与自由布局
- 精美时光动态背景、静谧纯白背景。
- 卡片支持自由拖拽排序。
- 支持双文件夹面板同时开启，实现卡片跨文件夹穿梭归类。

## 🛠️ GitHub 版本安装方法 (手动安装)

如果您想抢先体验最新代码，请按照以下步骤手动安装：

1. 前往 [**Releases**](https://github.com/majorxwho/STab/releases) 页面下载 `STab-v3.2.3.zip`。
2. 解压下载的 ZIP 文件到本地文件夹。
3. 在浏览器地址栏输入：
   - **Chrome**: `chrome://extensions/`
   - **Edge**: `edge://extensions/`
4. 开启页面右上角的“**开发者模式**”。
5. 点击“**加载解压的扩展程序**”，选择您刚才解压的文件夹即可。
